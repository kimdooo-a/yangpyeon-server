### [3] Windows Credential Manager UI 부재 진단 + Git 직접 우회 발견

> **사용자**: "A .... windows 시작 -그 어디에도 자격 증명 관리자가 없음."

Windows 11 검색이 자격 증명 관리자를 노출하지 않는 환경 (사용자 환경 특성). Run dialog (`control /name Microsoft.CredentialManager`) / Control Panel 등 우회 경로 다수 존재하지만, **Git 직접 명령** 이 더 깔끔.

진단:
- `cmdkey /list | findstr /i "github"` → 빈 결과 (한국어 Windows 인코딩 + Bash → cmd 호출 변환 문제)
- `cmdkey /list` 단독 → 한국어 인코딩 garbled "잘못된 인수" 에러 (Bash 환경에서)
- `git config --get credential.helper` = **`manager`** (= Git Credential Manager, GCM)

→ GCM 은 Windows Credential Manager (WCM) 을 storage backend 로 사용하는 어댑터. **`git credential reject`** 명령으로 GCM 을 통해 WCM entry 를 삭제 가능 (UI 우회).

**결론**: `printf "protocol=https\nhost=github.com\n\n" | git credential reject` 로 진행. Git credential helper 표준 프로토콜이라 cross-platform 안전.
