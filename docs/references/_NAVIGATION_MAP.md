## 라우트 맵

| 경로 | 페이지명 | 접근 권한 | 진입점 (어디서 오는가) | 이동 대상 (어디로 가는가) |
|------|----------|-----------|----------------------|-------------------------|
| `/` | 홈/랜딩 | 전체 | 직접 접근, 로고 클릭 | `/login`, `/signup`, `/features` |
| `/login` | 로그인 | 비로그인 | GNB, 홈 CTA, 권한없음 리다이렉트 | `/signup`, `/forgot-password`, → `/dashboard` |
| `/signup` | 회원가입 | 비로그인 | GNB, 로그인 페이지 | `/login`, → 이메일 인증 안내 |
| `/forgot-password` | 비밀번호 찾기 | 비로그인 | 로그인 페이지 | `/login` |
| `/dashboard` | 대시보드 | 로그인 | 로그인 성공 리다이렉트, GNB | 각 기능 페이지 |

## 네비게이션 구조

### GNB (Global Navigation Bar)

```
[비로그인 상태]
로고(→/) | 기능소개(→/features) | 요금제(→/pricing) | 로그인(→/login) | 회원가입(→/signup)

[로그인 상태]
로고(→/) | 대시보드(→/dashboard) | [프로필 드롭다운 ▼]
                                    ├─ 내 프로필(→/profile)
                                    ├─ 설정(→/settings)
                                    └─ 로그아웃(→/)
```

### 사이드바 (로그인 후, 해당 시)

```
대시보드(→/dashboard)
├─ [기능1](→/feature1)
├─ [기능2](→/feature2)
├─ [기능3](→/feature3)
└─ 설정(→/settings)
```

### Footer

```
서비스 소개(→/about) | 이용약관(→/terms) | 개인정보처리방침(→/privacy) | FAQ(→/faq) | 문의(→/contact)
```

## 리다이렉트 맵

| 트리거 | 출발 | 도착 | 조건 |
|--------|------|------|------|
| 로그인 성공 | `/login` | `/dashboard` | returnUrl 있으면 해당 URL |
| 회원가입 완료 | `/signup` | `/verify-email` | - |
| 로그아웃 | 어디서든 | `/` | - |
| 비로그인 접근 | 보호된 페이지 | `/login?returnUrl=원래경로` | - |
| 404 | 존재하지 않는 경로 | `/404` | 홈으로 돌아가기 링크 포함 |
