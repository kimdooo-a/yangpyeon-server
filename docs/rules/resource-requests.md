# 외부 리소스 요청 규칙

> 버전: 2.0 | 최종 업데이트: 2026-01-29
> 요청 형식 정의만 포함. 예시는 별도 파일 참조.

---

## 이미지가 필요할 때

Claude Code는 이미지를 직접 생성할 수 없음.
이미지 필요 시 사용자에게 아래 형식으로 요청:

```
🖼️ 이미지 요청

용도: (어디에 사용할 이미지인지)
크기: (예: 1200x630, 512x512)
형식: (PNG/JPG)
스타일: (예: 미니멀, 일러스트, 사실적)

[Antigravity 프롬프트]
(영어로 작성된 이미지 생성 프롬프트)

📁 저장 정보
- 위치: public/images/[카테고리]/
- 파일명: [파일명].[확장자]
- 전체 경로: public/images/[카테고리]/[파일명].[확장자]

⚠️ 생성 후 위 경로에 정확히 저장해주세요.

[대체 옵션]
- Midjourney: (프롬프트)
- DALL-E: (프롬프트)
```

## 이미지 프롬프트 생성 요청 시

사용자가 "이미지 프롬프트 만들어줘" 요청하면:

1. **기존 체계 확인** (기존 프로젝트인 경우): 폴더 구조, 파일명 패턴, 색상 테마
2. **프로젝트 맥락 파악**: package.json, tailwind.config, globals.css, README.md
3. **색상 추출**: CSS 변수 → 색상명 변환
4. **용도별 프롬프트 생성**: 템플릿 활용

→ 상세 프롬프트 템플릿: [`image-prompt-templates.md`](./image-prompt-templates.md)
→ 완전한 예시: [`resource-requests-examples.md`](./resource-requests-examples.md)

---

## 프롬프트 작성 가이드

- 구체적이고 명확하게 작성
- 스타일, 분위기, 색상 명시
- 부정 프롬프트 활용 (avoid, without)
- AI 이미지에 텍스트 포함 금지 (부정확함) → 텍스트는 코드로 오버레이

## API 키/인증 정보 필요 시

```
🔑 인증 정보 필요

서비스: (예: Supabase, OpenAI)
필요한 값: (예: SUPABASE_URL, SUPABASE_ANON_KEY)
설정 위치: .env.local

사용자 조치:
1. [서비스] 대시보드 접속
2. [경로]에서 키 복사
3. .env.local에 추가
```

## 외부 파일/데이터 필요 시

```
📁 외부 파일 필요

파일 종류: (예: CSV 데이터, 디자인 시안)
용도: (어디에 사용할지)
권장 형식: (예: CSV, JSON, PNG)
저장 위치: (예: /public/data/)
```

## 브라우저 테스트 필요 시

```
🌐 Antigravity 브라우저 테스트 요청

테스트 URL: http://localhost:3000/페이지
확인 항목:
- [ ] 항목1
- [ ] 항목2

스크린샷 필요: 예/아니오
```

---

## 요청 시 주의사항

- 왜 필요한지 맥락 설명
- 구체적인 스펙 제시 (크기, 형식)
- **저장 경로와 파일명을 반드시 명시**
- 대안이 있으면 함께 제시
- 급하지 않으면 "선택사항" 명시
