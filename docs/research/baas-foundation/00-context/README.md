# 00 — Context (BaaS Foundation 작업의 사전 분석)

이 디렉토리는 ADR-022~029 (멀티테넌트 BaaS 전환) 작업을 시작하기 전에 수집한 사전 분석 결과를 담습니다.

## 작업 트리거

**2026-04-26 사용자 요구사항**:
> "나혼자 나의 프로젝트 10~20개를 운영하고자하며, 그 백엔드로 이 프로젝트를 사용하려고해. 그래서 개발 부담이 있더라도 이 프로젝트가 내가 사용하고 운영하는 프로젝트들의 백엔드로서 계속적인 확장이 가능하도록 근본적인 설계방향이 필요해."

이 요구사항은 ADR-001 (Multi-tenancy 의도적 제외)의 재검토 트리거 4개 중 다음에 해당:
- 트리거 1: 사용자/조직 2개 이상 6개월 이상 지속 (= 10~20개 프로젝트의 영구 운영)
- 트리거 3: 독립 팀/조직 관리 FR 신규 추가

따라서 ADR-001 재검토 + ADR-022~029 신설이 정당화됨.
