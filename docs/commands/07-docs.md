# 7. 문서/파일 관리

## 7-1. CLAUDE.md 정리

[사용자 → Claude Code]
```
CLAUDE.md가 비대해졌어.
긴 섹션을 docs/rules/에 분리해줘.
CLAUDE.md에는 참조 링크만 남기고.
```

---

## 7-2. rules 파일 정리

[사용자 → Claude Code]
```
docs/rules/ 파일들 확인해줘.
비대한 파일 있으면 분할하거나 정리해줘.
_index.md도 업데이트해줘.
```

---

## 7-3. 규칙 파일 현황 확인

[사용자 → Claude Code]
```
docs/rules/_index.md 보여주고
각 규칙 파일 요약해줘.
```

---

## 7-4. status history 정리

[사용자 → Claude Code]
```
docs/status/current.md가 비대해졌어.
현재 내용을 docs/status/history/YYYY-MM.md에 추가하고
current.md는 새로 시작해줘.
```

---

## 7-5. logs 정리

[사용자 → Claude Code]
```
docs/logs/YYYY-MM.md가 비대해졌어.
다음 월 파일 생성해줘.
```
